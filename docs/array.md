### Usage
just import the file before using.

```js
import { /* methods */ } from 'dewjs/array';
```

##
### contain
(***source*** : *Array*, ***value*** : *Any*) => *Object* | *Boolean (false)*

- **source*
- **value*

##
### subtract
(***source*** : *Array*, ***target*** : *Array*) => *Array*

- **source*
- **target*

##
### difference
(***source*** : *Array*, ***target*** : *Array*) => *Array*

- **source*
- **target*

##
### compare
(***source*** : *Array*, ***target*** : *Array*) => *Array*

- **source*
- **target*

##
### unique
(***source*** : *Array*) => *Boolean*

- **source*

##
### natSort
(***source*** : *Array*, ***settings*** : *Array*) => *Array*

- **source*
- *settings*

settings:

- *inside*
- *reverse*

- **value*

- *caseSens*
